2024 {六角學院}體驗營感謝心得牆


報名動機

事隔兩年，我來二刷六角的體營驗囉…！
正逢轉職中，一方面做 Side Project 外，另一方面重新復習前端技能，才不會荒廢生疏。而且這次多了 Hexo 架站工具，可以打造個人品牌網站，還有多頁 UI 設計稿，光聽就很心動，報名費用 $799，就立馬報名參加囉！

課程內容及資源

  • 5 場直播教學，5 月的每週三晚上
  • 60 小時超過 150 部的前後端程式影音課程，提供 6 個月期間無限次數觀看
  • 活動期間線上程式助教團隊，批改程式碼及作業
  • Discord 線上討論，講師，助教及同學一同幫助學習
  • 提供一式 UI RWD 設計稿共計 10 頁及 hexo 架站任務
  • 提供每週小組任務及每日任務

內容一樣多到爆表，而且是手把手一步一步帶你入門，真是超級用心~~

感想與心得

感謝洧杰校長的爆肝教學，將各單元拆成小節，方便再次複習，及不斷加碼(加碼課程都是目前最實用的技術)，在RWD及JQuery的應用上耐心又細心的拆解功略，除了知識的教學分享外，也著重輸出還設計每週小組任務及每日任務供學員練習，也感謝各位用心的助教沒日沒夜的批改各項作業，並給予超多的改善建議及鼓勵，感謝各位學員及前輩在 Discord 上熱烈瘋狂討論各種解題上的技巧，卡關部份都能在討論群組上找到解答。這次的體驗營還學到了 markdown 語法技巧，也完成1500團戰任務，影片延長一年觀看….真是收穫不少^^。

最後要感謝自己的不放棄，才能完成這次的挑戰.

最後再來回顧一下六角洗腦箴言…XD

  • 跟自己比較
  • 你不用一開始就很厲害，但要開始才有辦法很厲害
  • 只要你不放棄自己，六角團隊也不會放棄你
